# RESTful API - PostgreSQL/ElephantSQL

Aplicação RESTful desenvolvida em Node/Express e integração com banco de dados PostgreSQL usando DBaaS [https://www.elephantsql.com/](https://www.elephantsql.com/).

### Endpoints

---

- GET - /users
- GET by id - /users/id
- POST - /users
- PUT - /users/id
- DELETE - /users/id
